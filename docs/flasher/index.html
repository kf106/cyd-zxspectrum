Cheap Yellow Display — ZX Spectrum firmware

Install the latest firmware in your browser. You need a USB data cable, Google Chrome or Microsoft Edge on a computer (not phone or tablet), and an ESP32-2432S028 (CYD) board.

  1. Plug the CYD into your computer with USB.
  2. Click Install firmware below and allow access to the serial port when asked.
  3. Pick the CYD’s USB serial port (often USB Serial or CP210x).
  4. Wait until flashing finishes. On first boot you will get touch calibration and left/right-handed choice.
This installer needs Chrome or Edge on a desktop/laptop (Web Serial is not available here). Open this page over HTTPS (for example via GitHub Pages), not as a local file.
Note: Installing firmware does not copy games onto the SD card. Use a FAT32 card and copy .tap, .tzx, .z80, or .sna files separately, or use Menu → Load game / tape after flashing.
